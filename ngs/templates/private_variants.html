Variants endpoint (private)

To obtain data you have to perform post request to this endpoint.

There are several options to ask for data.

You can ask about variants in specific place.
Sample POST request using curl:

curl -d '{"chrom":"1", "start":"1"}' -H "Content-Type: application/json" -X POST post_url

The other option is to ask about variants in a specific range.
Sample POST request using curl:

curl -d '{"chrom":"1", "start":"1", "end":"10"}' -H "Content-Type: application/json" -X POST post_url

The nest option is to ask for a whole chromosome variants.
Sample POST request using curl:

curl -d '{"chrom":"1"}' -H "Content-Type: application/json" -X POST post_url
